# ▶️ Cara Menjalankan Notebook

## Menggunakan Google Colab

1. Buka notebook yang diinginkan dari repository GitHub.
2. Klik **Open in Colab** atau upload notebook ke Google Colab.
3. Jalankan seluruh cell secara berurutan.

## Menjalankan Secara Lokal

Clone repository:

```bash
git clone https://github.com/imaisnaini/data-science-2026.git
```

Masuk ke folder project:

```bash
cd data-science-2026
```

Install library yang diperlukan:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn scipy
```

Jalankan Jupyter Notebook:

```bash
jupyter notebook
```
